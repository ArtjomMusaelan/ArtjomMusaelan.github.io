Project 07

Emotion Detection System (Content Intelligence Agency - Phase 1)

NLP Engineer Team of 3 71% F1 Score DistilRoBERTa Multi-Language 7 Emotions Fine-Tuned Transformers XAI
Overview

Multi-language emotion analysis for unscripted TV

Developed an NLP pipeline for Content Intelligence Agency to automate emotion detection in unscripted television programs across multiple languages.

The system processes YouTube videos through speech-to-text transcription, machine translation, and emotion classification using fine-tuned DistilRoBERTa, achieving 71% weighted F1 score on domain-specific TV show data with 7 emotion categories.

The Challenge

Adapting NLP models to spoken television language

Content Intelligence Agency needed an automated system to identify emotions in unscripted TV shows (Survivor, Hunted, Big Brother) across multiple languages, but existing solutions struggled with spoken language characteristics.

The challenge required handling informal grammar, interruptions, filler words, and regional dialects while maintaining accuracy across languages. The system needed domain adaptation from written text to conversational TV language with explainable predictions for client trust.

Technical Implementation

End-to-end multi-stage NLP pipeline

Speech-to-Text & Translation Pipeline

  • — Integrated AssemblyAI and Whisper APIs for multi-language video transcription
  • — Conducted rigorous Word Error Rate (WER) evaluation by manually annotating 60-80 lines per team member
  • — Fine-tuned mBART translation model for language-to-English translation using OPUS datasets
  • — Manually evaluated translation quality on 150+ sentences using 3-point scoring system

NLP Feature Engineering

  • — Extracted Part-of-Speech tagging using SpaCy for linguistic features
  • — Applied TF-IDF vectorization for term importance analysis
  • — Implemented sentiment analysis using VADER and TextBlob
  • — Utilized pretrained word embeddings (Word2Vec, GloVe) and trained custom domain-specific embeddings

Model Development & Fine-Tuning

  • — Model Comparison: Evaluated 7+ architectures including Logistic Regression, Naive Bayes, SVM, LSTM, RNN, BERT variants
  • — Fine-Tuning: DistilRoBERTa with 1.9M parameters on 86,224 samples from GoEmotions and domain-specific data
  • — Dataset: Combined GoEmotions, collaborative team data, and 1,042 company-specific TV sentences
  • — Error Analysis: 750+ word report identifying misclassification patterns (neutral over-prediction, surprise-happiness confusion)
  • — Prompt Engineering: Explored few-shot prompting with locally hosted LLMs
  • — Cross-Validation: Robust evaluation ensuring generalization across emotion categories

Explainable AI Implementation

Implemented cutting-edge XAI techniques to ensure model transparency and build client trust in predictions.

  • — Layer-wise Relevance Propagation: Implemented with Conservative Propagation technique
  • — Token-Level Visualization: Displayed relevance scores showing which words influenced predictions
  • — Verification: Confirmed model focuses on emotionally charged words rather than irrelevant features
  • — Client Trust: Provided interpretable predictions for production deployment
