Project 06

Plant Root Segmentation (NPEC)

ML & Computer Vision Engineer 82.4% F1 Score U-Net + Graph Algorithms Deep Learning Production Pipeline
Overview

Production computer vision for high-throughput plant phenotyping

Developed a extensive computer vision solution for NPEC's Hades plant phenotyping system to automate root segmentation and measurement of Arabidopsis thaliana seedlings at scale.

This end-to-end pipeline addresses the challenge of analyzing thousands of plants daily for plant-microbe interaction research, enabling NPEC's liquid handling robot to inoculate plants at precise root tip locations with pixel-perfect accuracy.

The Challenge

Scaling agricultural research with automation

NPEC's high-throughput phenotyping facility needed automated root analysis to support their liquid handling robot that inoculates plants at precise root tip locations. Manual measurement of root systems was time-consuming and couldn't scale to their capacity.

The challenge was building a system capable of processing 10,000+ seedlings across 2,000+ Petri dishes, combining traditional computer vision for instance segmentation with deep learning for semantic segmentation, and extracting root system architecture using graph-based algorithms.

Technical Implementation

End-to-end pipeline combining multiple CV techniques

Stage 1: Data Preparation & Traditional CV

  • — Petri dish detection achieving ±30 pixel accuracy using traditional computer vision techniques
  • — Instance segmentation separating individual plants within each Petri dish
  • — Manual annotation workflow creating ground truth labels for training data
  • — Data cleaning and preprocessing ensuring quality training datasets

Stage 2: Deep Learning Segmentation

  • — Trained U-Net model with 1.9M parameters on 36,663 patches from complete training dataset
  • — Achieved 82.4% F1 score on 5,445 validation patches, exceeding 80% target
  • — Implemented patch-based inference for memory management on large images
  • — Applied morphological post-processing: small object removal, binary closing, Gaussian smoothing

Stage 3: Root System Architecture Extraction

  • — Graph-Based Extraction: Used NetworkX and skeleton topology to distinguish primary from lateral roots
  • — Dijkstra's Algorithm: Applied for primary root identification with vertical orientation filtering
  • — Skeleton Analysis: Converted segmented regions to skeletal representations for topological analysis
  • — Measurement Pipeline: Outputs primary root lengths for robotic liquid handling systems
  • — Multi-Dataset Robustness: Handled varying image characteristics across different experiments
  • — Production Integration: Complete 8-stage pipeline from raw images to measurements

Key Technical Innovations

  • — Graph traversal algorithm distinguishing root types using topology
  • — Patch-based inference managing memory for large-scale processing
  • — Multi-stage pipeline combining traditional CV with deep learning
  • — Pixel-perfect accuracy for robotic system integration
