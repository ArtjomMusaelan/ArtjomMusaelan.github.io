Project 04

Safer Roads in Breda (ANWB)

Data Scientist Team of 5 XGBoost Model Streamlit Deployment
Overview

Predicting street-level road safety risk for ANWB

Developed a production-ready machine learning application for ANWB that predicts street-level road safety risk in Breda, supporting the organization's mission of "preventing instead of insuring."

As the Data Scientist on Team No. 6, I was responsible for the complete ML pipeline from model selection through deployment, including XGBoost development, optimization, and contributing to the Streamlit application interface.

The Challenge

Building localized, predictive safety features

The Netherlands has invested heavily in road safety programs, reducing road deaths by 1600-1700 between 1988-2007. However, existing navigation apps lack localized, predictive safety features.

The challenge was integrating multiple datasets (ANWB Safe Driving 2017-2023, KNMI Weather, Bron Accidents) to create street-specific risk predictions that combine historical incident data with real-time weather conditions for proactive route planning.

My Role & Technical Approach

Leading ML development in collaborative team environment

My Core Responsibilities (Data Scientist)

  • — Designed and implemented XGBoost classification models for street-level risk prediction
  • — Developed iterative modeling pipeline with Proof-of-Concept (POC) validation approach
  • — Performed comprehensive model evaluation using RMSE, Recall, and MAE metrics
  • — Optimized hyperparameters using GridSearchCV for best performance
  • — Trained multiple model variants (with/without coordinate data) to prevent overfitting
  • — Contributed to Streamlit deployment for real-time predictions

Data Integration & Feature Engineering

  • — Combined 3 major datasets: ANWB Safe Driving (2017-2023), KNMI Weather data, and Bron Accidents
  • — Created temporal features accounting for date/time patterns in incidents
  • — Developed weather categorizations incorporating wind, precipitation, and temperature
  • — Engineered incident severity groupings across 4 categories: Speed, Harsh Cornering, Harsh Accelerating, Harsh Braking
  • — Implemented location-based features using latitude/longitude coordinates
